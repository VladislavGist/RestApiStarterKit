This is REST api

Technology stack:
    - yarn
    - es6
    - nodejs
    - express 4
    - passportjs
    - mongodb

Installing project:
    1. install nodejs > 7.06.x
    2. install mongodb
    3. yarn install

Control project:
    start server in production - yarn run prod:start
    start dev server - yarn run dev:start
    build project - yarn run build:start
